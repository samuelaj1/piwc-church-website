# PIWC Hull Website

Official website for **Pentecost International Worship Centre (PIWC) Hull**, a branch of **The Church of Pentecost – UK**. This website serves as a digital platform to connect worshippers, share church activities, livestreams, sermons, events, ministries, and provide information about the church community.

## 🌍 Overview

The PIWC Hull website is designed to create a welcoming online presence for members, visitors, and the wider community. It provides easy access to church information, worship schedules, ministries, announcements, sermons, media content, and opportunities to support the ministry.

## ✨ Features

* Modern and responsive church website design
* Hero sliders with church branding
* About PIWC Hull section
* Service times and worship information
* Sermons and media section
* Events and announcements
* Ministries and church departments
* Donation / Support section
* Contact information and location
* Mobile-friendly user interface
* Church-themed blue & gold branding

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/piwc-hull-website.git
```

### Open the project

Navigate to the project folder and open `index.html` in your browser.

Or run with a local server:

```bash
# VS Code Live Server
Right click index.html → Open with Live Server
```
